Release 2.9.8

New features:

  • The client now displays groups defined in the web application of digital-stage, sorted alphabetically.
  • Each groups now has a summary volume control allowing you to change the volume of all group members at once.
  • Metronome - there is now a server-side metronome that can be started and stopped by everyone, and set for a count-in of a predefined numbers of measures and beats.
  • Local monitor is now default on first startup, as is monitor off. To turn on local monitor signal, you need to switch from "Send" to "Mono".
  • Monitor controls are only shown if at least one channel has monitor signal turned on.
  • When accepting an invite for a stage of type "web", a link is offered to open the browser with that stage.

Bugfixes:

  • Some stability fixes in case of bad server connectivity.
